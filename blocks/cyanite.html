Cyanite and Grafana

A modern alternative to the graphite project

cyanite and grafana

What does cyanite do ?

Cyanite exposes a simple TCP service to input metrics from the network. Cyanite implements the carbon protocol as designed by the graphite project. Apache Cassandra is used to store metric data.

What does grafana do ?

An open source, feature rich metrics dashboard and graph editor for Graphite & InfluxDB.

Grafana is a graphite frontend providing rich graphing options and dashboards.
